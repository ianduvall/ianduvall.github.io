Satellite Remote Sensing Library

I am a member of a University of Michigan Multidisciplinary Design Team that is building a standard interface template library for satellite image processing research. I built the standard interface template layer that takes advantage of Object Oriented Programming as well as design patterns like the Abstract Factory pattern. The standard interface layer abstracts the different implementations that are necessary when working with multiple data format types.

ARTDES 193 Project

This website was started as a project for a class at the University of Michigan's Art and Design School. This project focused primarily on front-end development which was my first formal introduction to the depths of HTML and CSS. Designing this website has shown me how front-end development is really an art form and I realized that I enjoy working on both the front-end and back-end. This is just the beginning of the development of this site so check back in future for cool updates!
